コンパイラ等にdebktProを使用
debktPro: https://sourceforge.net/projects/devkitpro/?source=typ_redirect

1. 何をするプログラムか
　シューティングとブロック崩しを遊ぶ事ができるプログラムです。
　タイトル画面でゲームを選択して遊びます。
　＜ブロック崩し＞
　　ボールをブロックに当てて、全てのブロックを消せばクリアです。
　跳ね返ってくるボールをバーに当てて、ボールを跳ね返しましょう。
　ボールを跳ね返せずに、画面下の画面外に落ちるとゲームオーバーです。

　＜シューティング＞
　 レーザーを撃って敵を倒すゲームです。全ての敵を倒せばクリアです。
　敵は合計3体おり、１体ずつ出て来ます。敵はレーザーを３発当てれば倒せます。
　ただし、１発当てるとしばらく(点滅している間)ダメージが入りません。
 * 敵は弾として、小さな敵を撃ってきます。小さな敵は倒せないので注意してください。
 
　 プレイヤーのライフは基本1なので、1発でも敵の弾・敵本体に当たるとゲームオーバーです。
　当たり判定は少し大きいので、余裕をもって避けてください。
　 また、スコアも存在し、スコアは以下の要因から決まります。
　・倒すのにかかった時間
　・打った弾数
　・ランダムに落ちてくる緑色のPと書かれたアイテムの取得数　
　アイテムは触れれば自動で回収します。

2. プログラムの使い方
＜タイトル画面＞
　十字キー（縦方向のみ）: 遊ぶゲームを選びます。選択しているゲームの隣にピンク色の三角形が表示されます。
　Aボタン: 選択していゲームを開始します。
　* どうしてもシューティングがクリアできない場合は、シューティングをスタートする時に、Aボタンと一緒に左十字キーを押してください。
　  残機が１増えて１発までは耐えられるようになります。ただし、スコアは下がります。

＜ゲーム内共通操作＞
　・ゲーム開始時・リトライ時に STARTボタン: ゲームを開始します。
　・ゲーム中に SELECTボタン: ゲームを一時中断(ポーズ画面に遷移)します。再度SELECTボタンを押せば再開します。
　・ポーズ画面時に L + Rボタン: タイトル画面に戻ります。
　・GAME OVERE画面時に STARTボタン: リトライできます。
　・CREAL画面時に STARTボタン: リトライできます。

＜シューティング＞
　・十字キー:　上下左右に自機を動かす事ができます。
　・Aボタン:　レーザーを撃つ事ができます。連打可能です
　・Bボタン + 十字キー: 速度２倍で動かす事ができます。 

＜ブロック崩し＞
・十字キー　左右の入力でラケットを横に動かします。上下はありません。

3. 資料外の技術
 3.1 使用した技術
　・スプライト
　・擬似乱数
 
 3.2 どこで使用したか
 ＜スプライト＞
  動かす物・点滅する物の描画に使いました。
  具体的には、シューティングの自機、敵、弾、アイテム、ブロック崩しのラケット、弾、選択画面の点滅する三角形。
 
 ＜擬似乱数＞
  シューティングのアイテムの出現率及び、出現座標をランダムに決定する事に使いました。
